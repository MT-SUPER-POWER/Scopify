## 📌 填写示例

### Bug 描述
歌手页切换「专辑」「相似歌手」Tab 后内容不刷新，仍然显示上一个 Tab 的数据。

### 复现步骤
1. 打开歌手页（如 `/artist?id=123`）
2. 默认显示「热门歌曲」Tab，数据正常
3. 点击「专辑」Tab
4. 内容区仍显示热门歌曲列表，未切换为专辑列表

### 期望行为
点击「专辑」Tab 后内容切换为专辑列表，点击「相似歌手」切换为相似歌手列表

### 实际行为
Tab 高亮状态切换了，但内容区不变，控制台无报错

### 环境信息

| 项 | 值 |
|---|---|
| 页面 URL | `/artist?id=123` |
| 浏览器 / Electron | Chrome 125 / Electron |
| 操作系统 | Windows 11 |
| 用户登录态 | 已登录 |

### 错误信息
```
（控制台无报错，Network 面板可见 `artist/albums` 请求正常返回 200）
```

- [ ] 阻断
- [ ] 严重
